# 🐱 AI Cats

A React web app that fetches and displays random cat images from an external API.

Users can filter images by limit, theme, and size.

---

## 🚀 Features

- Random cat images from API  
- Filter options (limit, theme, size)  
- Loading state while fetching data  
- Error handling for API requests  
- Clickable images for detail view  

---

## 🛠️ Technologies

- React  
- JavaScript  
- HTML  
- CSS  
- REST API  

---

## ▶️ How to run

npm install  
npm start
